# Cloq

Stealthy floating AI & WhatsApp assistant for macOS.

---

## Installation

### Option 1: Recommended (Two-Step)
```bash
brew tap CloqApp/cloq-release https://github.com/CloqApp/cloq-release
brew install --cask cloq
````

### Option 2: One-Liner

```bash
brew tap CloqApp/cloq-release https://github.com/CloqApp/cloq-release && brew install --cask cloq
```
